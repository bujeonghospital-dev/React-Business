======================================================================

6. ERROR RESPONSES
======================================================================

กรุณาแจ้ง Error Codes และความหมาย:

HTTP Status     Error Code      Description
-----------------------------------------------------------------------
400            INVALID_PHONE    เบอร์โทรไม่ถูกต้อง

403            FORBIDDEN        ไม่มีสิทธิ์ใช้งาน

500            SERVER_ERROR     เกิดข้อผิดพลาดที่ server

{
  "success": false,
  "error": "INVALID_PHONE",
  "message": "Phone number format is invalid",
  "details": "Phone number must be 10 digits starting with 0"
}

8. CHECK CALL STATUS API
======================================================================

เรามี API สำหรับเช็คสถานะการโทรหรือไม่?

GET /v1/robocall/{call_id}/status

Response:
{
  "call_id": "robocall-123456",
  "status": "in_progress",
  "phone_number": "0812345678",
  "start_time": "2025-11-07T10:00:00.000Z",
  "answer_time": "2025-11-07T10:00:15.000Z",
  "end_time": null,
  "duration": 15,
  "result": null
}


Status Values:
- pending: รอดำเนินการ
- dialing: กำลังโทร
- ringing: กำลังดังสาย
- answered: รับสายแล้ว
- busy: ไลน์ไม่ว่าง
- no_answer: ไม่รับสาย
- failed: โทรไม่สำเร็จ
- completed: จบสาย
